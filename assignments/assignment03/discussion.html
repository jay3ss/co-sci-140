1. How to manipulate data using a structure

    Before being used, structures (or, structs) must be declared. This is done in the following example 

    struct Student // structure tag
{
    int studentID;      // structure member
    string name;        // structure member
    short yearInSchool; // structure member
    double gpa;         // structure member
};

    The creation of a struct variable of has similar semantics to that of primitive data types. And, to access the struct's members we use the dot (.) operator. When initializing a struct, you have a couple of options. For example 

    // Dot notation
Student stu1;               // similar to int myInt;
stu.studentID = 1234567890;
stu.name = "Billy";
stu.yearInSchool = 1;
stu.gpa = 3.3;

// Array-like notation
Student stu2 = {0987654321, "Johnny", 4, 2.7};

// Partial initialization
Student stu3 = {2358132134, "Amelia", ,};

    NOTE: If you leave one member uninitialized, you must leave all other members that come after it uninitialized as well.

    Accessing member data is straightforward using the dot notation

    cout << stu1.studentID
     << stu1.name
     << stu1.yearInSchool
     << stu1.gpa;

    Just like built-in data types, you can make pointers to structs. The creation of pointers to structs is similar to pointers of any other type, but you can access the struct's members using two different methods. As you'll see, one is easier to read than the other. 

    // Dereferencing the pointer
*(stu).name = "Billy";

// Arrow notation
stu->gpa = 3.3;

  2. How arrays can be used in a structure

    structs' members can include arrays, but you must be careful in how you declare them. Member arrays must have their length defined in the definition of the struct unless it's the last member of the struct [1]. For example, 

    // Will compile, flexible length member array declared
// as last member
struct Testing1
{
    int number;
    int numbers[];
};

// Won't compile, member array length not declared
struct Testing2
{
    int numbers[];
    int number;
}

// Will compile, member array length declared
struct Testing3
{
    int numbers[5];
    int number;
};

    To use an array member, you use it just like any other array.

    Testing3 testing;
for (int i = 0; i < 5; i++)
{
    testing.numbers[i] = i;
}

  3. How to create an array of structures

    Arrays of structures can be created quite easily, just like an array of built-in data types. Arrays of structs can also be dynamically allocated. For example, 

    const int NUM_STUDENTS = 50;
Student students1[NUM_STUDENTS];

// Dynamically allocate enough space for 50 students
Student *students2 = nullptr;
students2 = new Student[NUM_STUDENTS];

    will create two arrays holding enough space for 50 Student type structs each.
