1. Characters:

    Characters, or the char data type, are used to store individual characters. A variable of type char can only hold one character at a time. chars use the single quote (') around the character (not double quotes, ").

    Example: 

    char letter = 'a';

    C-Strings:

    A sequence of characters stored in adjacent memory locations (e.g., an array of chars) that's terminated by the NULL character.

    Example: 

    char cstr[7] = {'H', 'e', 'l', 'l', 'o', '!', '\0'};
    where \0 is the null terminator. Although, you can create a C-string with an implied size (and the null terminator will be added automatically) by using a string literal as shown below 
    char cstr[] = "Hello!";

    Strings:

    The string class is a data type (not built-in to C++) that allows you to work with string literals. Strings, similar to C-strings, are a series of characters stored in consecutive memory locations, and can be of almost any length. To use the string class, you need to include the string header. Note that strings use double quotes (").

    Example: 

    string myString = "Hello!";

  2. The char data type, when not used as an array, is only a single character. A C-string is similar to a char array, except that a C-string is terminated with a null terminator [1]. Strings are similar to C-strings in that both are stored in consecutive memory locations and are terminated with the null terminator. Strings look like characters, but they are represented differently. Strings end with the null terminator while characters do not. All string literals are stored as C-strings, but the string class comes with some nice features that make them easier to work with compared to C-strings such as: flexible storage capacity, easier notation, easier comparing of strings (overloaded comparison operators), and concatenation [1].
